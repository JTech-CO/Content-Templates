# 금지 표현 및 대체 표현 모음

## 사용 목적

블로그, 쓰레드, 트위터 글에서 과장되거나 위험하게 읽힐 수 있는 표현을 줄이고, 더 자연스럽고 신뢰감 있는 문장으로 바꾸기 위한 참고용 문서입니다.

---

## 1. 공통 금지 표현

| 피해야 할 표현 | 대체 표현 |
| --- | --- |
| 무조건 추천합니다 | 이런 분께 잘 맞을 것 같습니다 |
| 반드시 해야 합니다 | 상황에 따라 고려해볼 만합니다 |
| 완벽합니다 | 전반적으로 만족스러운 편입니다 |
| 단점이 없습니다 | 큰 불편은 적었지만, 사용 환경에 따라 다를 수 있습니다 |
| 최고입니다 | 인상적이었던 부분이 있었습니다 |
| 역대급입니다 | 기대보다 괜찮게 느껴졌습니다 |
| 실패 없습니다 | 선택 전에 조건을 확인해보면 좋습니다 |
| 누구에게나 좋습니다 | [추천 대상]에게 특히 잘 맞을 수 있습니다 |

---

## 2. 제품 리뷰 대체 표현

| 피해야 할 표현 | 대체 표현 |
| --- | --- |
| 사야 합니다 | 구매를 고민 중이라면 비교해볼 만합니다 |
| 가성비 최고입니다 | 가격 대비 만족도가 괜찮은 편입니다 |
| 효과가 바로 나타납니다 | 사용 후 이런 점이 체감되었습니다 |
| 품질이 압도적입니다 | 마감이나 사용감에서 장점이 느껴졌습니다 |
| 단점이 전혀 없습니다 | 아쉬운 점은 크지 않았지만, [조건]은 확인이 필요합니다 |
| 무조건 재구매합니다 | 상황이 맞는다면 다시 구매할 의사가 있습니다 |

---

## 3. 체험단 / 협찬 글 대체 표현

| 피해야 할 표현 | 대체 표현 |
| --- | --- |
| 광고지만 진짜 좋아요 | 제공받은 제품이지만 실제 사용감을 중심으로 정리했습니다 |
| 협찬이라 좋게 쓰는 게 아닙니다 | 체험 과정에서 느낀 장점과 아쉬운 점을 함께 적었습니다 |
| 꼭 써보세요 | 이런 상황에서 사용해보면 좋을 것 같습니다 |
| 효과 보장합니다 | 개인적인 사용감 기준으로는 이렇게 느껴졌습니다 |
| 단점이 없습니다 | 사용하면서 아쉬웠던 부분도 함께 정리했습니다 |

---

## 4. 식당 / 맛집 글 대체 표현

| 피해야 할 표현 | 대체 표현 |
| --- | --- |
| 인생 맛집입니다 | 다시 방문하고 싶을 만큼 만족스러웠습니다 |
| 무조건 웨이팅할 가치가 있습니다 | 대기 시간이 있어도 고려해볼 만한 곳입니다 |
| 누구나 좋아할 맛입니다 | 호불호가 크지 않을 만한 맛으로 느껴졌습니다 |
| 진짜 미쳤습니다 | 맛과 구성이 인상적이었습니다 |
| 여기 안 가면 손해입니다 | 근처에서 식사할 곳을 찾는다면 참고할 만합니다 |

---

## 5. 여행 글 대체 표현

| 피해야 할 표현 | 대체 표현 |
| --- | --- |
| 무조건 가야 하는 곳입니다 | 일정이 맞는다면 들러볼 만한 곳입니다 |
| 평생 잊지 못할 여행지입니다 | 분위기가 오래 기억에 남는 장소였습니다 |
| 누구에게나 완벽한 코스입니다 | [여행 목적]에 특히 잘 맞는 코스입니다 |
| 후회 없습니다 | 기대하는 여행 스타일과 맞는지 확인하면 좋습니다 |
| 사진이 무조건 잘 나옵니다 | 시간대와 위치에 따라 사진 찍기 좋은 포인트가 있습니다 |

## 6. 아티클 / 정보성 글 대체 표현

| 피해야 할 표현 | 대체 표현 |
| --- | --- |
| 확실히 바뀔 것입니다 | 변화 가능성을 관찰할 필요가 있습니다 |
| 업계가 뒤집힙니다 | 업계에 영향을 줄 수 있는 변수로 볼 수 있습니다 |
| 모두가 주목합니다 | 관련 업계에서 관심이 커지고 있습니다 |
| 실패할 수밖에 없습니다 | 여러 제약 요인을 함께 살펴볼 필요가 있습니다 |
| 성공은 시간문제입니다 | 성과 여부는 향후 지표를 통해 확인해야 합니다 |

## 7. 건강 / 뷰티 / 식품 관련 주의 표현

| 피해야 할 표현 | 대체 표현 |
| --- | --- |
| 효과가 있습니다 | 개인적으로는 이런 사용감을 느꼈습니다 |
| 치료됩니다 | 제품 설명과 사용 경험을 기준으로 정리했습니다 |
| 살이 빠집니다 | 식단과 생활 습관에 따라 체감은 다를 수 있습니다 |
| 피부가 좋아집니다 | 사용 후 촉촉함이나 사용감이 이렇게 느껴졌습니다 |
| 누구에게나 안전합니다 | 개인 상태에 따라 다를 수 있으니 성분 확인이 필요합니다 |

---

## 8. 최종 적용 지시

글을 작성하거나 수정할 때 아래 원칙을 적용합니다.

- 과장, 단정, 보장, 강요 표현을 줄입니다.
- 실제 경험과 확인 가능한 사실을 중심으로 씁니다.
- 장점과 아쉬운 점을 함께 제시합니다.
- 독자가 스스로 판단할 수 있도록 조건과 맥락을 함께 제공합니다.
- 광고, 협찬, 제공 사실은 숨기지 않고 명확하게 표기합니다.
